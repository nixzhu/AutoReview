# AutoReview

AutoReview for app review.

## Requirements

Swift 3.0, iOS 8.0

(Swift 2.3, use version 0.2.0)

## Usage

See the demo.

## Installation

### Carthage

```ogdl
github "nixzhu/AutoReview" >= 1.0.0
```

### CocoaPods

```ruby
pod 'AutoReview', '~> 1.0.0'
```
